SOBRE O PROJETO:
O objetivo da nossa equipe é desenvolver um software que, que a partir de dados fornecidos pelas empresas e organizações que utilizarem este produto, seja capaz de gerenciar os estoques de alimento com a intenção de reduzir a compra de quantidades desnecessárias, que acabarão, posteriormente, sendo descartadas. Através da análise de vendas, previsões de demanda e a data de validade dos produtos, o sistema será capaz de fornecer informações precisas sobre a quantidade ideal para as empresas adquirirem.

Este programa apresentado é um protótipo de nossa ideia. Ele foi desenvolvido pensando na fácil compreensão do código e com uma interface contendo uma linguagem mais humana, simulando um verdadeiro assistente virtual.

INSTRUÇÃO BÁSICA DE USO:
Ao instalar a pasta do projeto, o usuário deverá acessar o arquivo chamado "greenstorage.py". Tendo aberto este arquivo, o programa será executado automaticamente. O usuário poderá interagir com o assistente através da digitação de informações pelo teclado de seu computador, conforme se pede pelo sistema.

REQUISITOS:
Para que seja possível a utilização do assistente, o usuário necessitará ter o Python 3 instalado em seu computador.

DEPENDÊNCIAS:
As únicas dependências é ter o Python 3, conforme dito anteriormente.
